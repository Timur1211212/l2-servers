📸 Как сохранить OG изображения

Инструкция:

1. Нажмите F12 (или Ctrl+Shift+I) для открытия DevTools

2. Нажмите Ctrl+Shift+P и введите Capture screenshot

3. Выберите Capture area screenshot

4. Выделите нужное OG изображение (1200x630)

5. Сохраните скриншот с соответствующим именем:

og-image.jpg — для главной

og-interlude.jpg — для Interlude

og-highfive.jpg — для High Five

og-classic.jpg — для Classic

og-essence.jpg — для Essence

og-download.jpg — для страницы скачивания

og-scryde.jpg — для Scryde

og-main.jpg — для Main

og-lu4.jpg — для LU4

og-asterios.jpg — для Asterios

og-eglobal.jpg — для Eglobal

og-legacy.jpg — для Legacy

6. Загрузите файлы в папку /public вашего сервера

💡 Альтернативный способ (Windows):

Используйте инструмент Ножницы (Win+Shift+S) — режим "Фрагмент экрана"

💡 Альтернативный способ (Mac):

Используйте Cmd+Shift+4 для выделения области
